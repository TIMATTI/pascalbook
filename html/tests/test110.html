Тест: Вычисление логических выражений

 1. Логическими являются такие данные и выражения...
которые могут принимать только два значения – True и False
которые могут принимать только два значения – Yes и No
 2. Переменные логического типа описываются с использованием стандартного типа
bool
boolean
 3. Размер выделяемой памяти под логическое данное
5 байт
1 байт
 4. Ответьте, каков будет результат выполнения программы:
1.    program zadacha7_1;
2.    var A, B, C, rezA: boolean;
3.    begin
4.        A:=true;
5.        B:=false;
6.        C:=false;
8.        rezA:=(A and C) or B;
9.        writeln('rezA= ', rezA);
10.    end.

rezA=True
rezA=False
