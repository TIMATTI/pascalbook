Тема: Процедуры и функции

В большинстве языков программирования предусмотренно использовать вспомогательный алгоритм как подпрограмму. Для работы подалгоритма необходимо присвоить имя и описать алгоритм по правилам языка Паскаль.

В случае использования исполнителя Чертёжник появляются случаи повторения однобразных действий. При использовании обычного метода написания программы. Программа становится громоздкой. Чтобы этого избежать используют процедуры и функции.

Для единообразия функции и процедуры называются подпрограммами.


Описание процедур и функций

Любая используемая в программе процедура или функция должна быть предварительно описана в разделе описаний.

Описание процедуры имеет вид:
1.    procedure <имя> (<список формальных параметров>);
2.    <раздел описаний>
3.       begin
4.          <операторы>
5.       end;

Описание функции имеет вид:
1.    function <имя> (<список формальных параметров>): <тип возвращаемого значения>;
2.    <раздел описаний>
3.       begin
4.          <операторы>
5.       end;

Операторы подпрограммы, окаймленные операторными скобками begin/end, называются телом этой подпрограммы.

Список формальных параметров вместе с окружающими скобками может отсутствовать. Он состоит из одной или нескольких секций, разделенных символом ";". Каждая секция состоит из списка переменных, перечисляемых через запятую, после которого следуют двоеточие и тип.

Пример 1: Процедура считающая сумму двух переменных.
1.    procedure Sum(x,y: real; res: real);
2.       begin
3.          res:=x+y;
4.       end;


Вызов подпрограммы

Подпрограмма один раз описывается и может быть многократно вызвана. Для вызова процедуры используется оператор вызова процедуры:

Пример 2: Использование процедуры из Пример 1 в основной программе.
1.    var
2.       a,b,res: real;
3.    begin
4.       readln(a,b);
5.       Sum(a,b,res);   //вызов прцедуры
6.       writeln(a,' + ',b,' = ',res);
7.       Sum(a*2,b,res);   //вызов прцедуры, где a=a*2
8.       writeln('(',a,'*2) + ',b,' = ',res);
9.       Sum(a,b+5,res);   //вызов прцедуры, где b=b+5
10.       writeln(a,' + (',b,'+5) = ',res);
11.    end.


Переменная Result

Внутри тела любой функции определена специальная переменная с именем Result, которая хранит результат вычисления функции. Ее тип совпадает с типом возвращаемого значения функции.
Пример 3: Функция считающая сумму(разность) двух чисел.
1.    {----фунция суммы----}
2.    function Sum(x,y: real): real;
3.       begin
4.          Result:=x+y;
5.       end;
6.    {----фунция разности----}
7.    function Sub(x,y: real): real;
8.       begin
9.          Result:=x-y;
10.       end;
11.    {----основная программа----}
12.    begin
13.       writeln('Функция суммы: ,Sum(2,3));
14.       writeln('Функция разности: ,Sub(2,3));
15.    end.

Если внутри функции не присвоить переменной Result некоторое значение, то функция вернет в результате своего вызова непредсказуемое значение.
