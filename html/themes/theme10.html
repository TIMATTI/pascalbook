Тема: Строки

До этого урока мы работали, используя в основном числовые типы данных: byte, integer, longint, real.

Для обработки текстов в Паскале широко используется тип string (строка). Количество символов в строке может меняться от 0 до 255.

Самый первый байт в строке имеет индекс 0 и содержит текущую длину строки. Первый значащий символ строки занимает второй байт и имеет индекс 1.

Вce действия над строками и символами реализуются с помощью описываемых ниже стандартных процедур и функций:

function Сoncat(S1, [S2,...,SN]) - возвращает строку, представляющую собой сцепление строк — параметров S1,S2,... ,SN.

function Copy(ST, INDEX, COUNT) - копирует из строки ST COUNT символов, начиная с символа с номером INDEX.

procedure Delete(ST, INDEX, COUNT) - удаляет COUNT символов из строки ST, начиняя с символа с номером INDEX.

procedure Insert(SUBST, ST, INDEX) - вставляет подстроку SUBST в строку ST, начиная с символа с номером INDEX.

function Length(ST) - возвращает длину строки ST.

Пример 1: Вводится строка ST. Вычислить её длину.
1.    program zadacha6_1;
2.    var
3.       st: string;
4.       n: integer;
5.    begin
6.       write('Введите строку: ');
7.       readln(st);
10.       n:=length(st);   //вычисление длины строки
8.       writeln('Длина строки:',n);
10.    end.

1.    program zadacha6_2;
2.    var
3.       t: string;
4.       b: string[1];   //[1] - максимально возможная длина строки
5.       n,k,i: integer;
6.    begin
7.       write('Введите текст: ');
8.       readln(st);
9.       n:=length(st);
10.       k:=0;
11.       for i:=1 to n do begin
12.          b:=copy(t,i,1);   //копирование одного символа
13.          if (b='a') or (b='A') or (b='а') or (b='А') then k+=1;
14.       end;
15.       writeln('k: ',k);
16.    end.
